Propósito del sitio

El proposito de este sitio es poner en practica los conocimientos adquiridos sobre HTML5 que corresponde a la version 5 de HTML- Hiper Text Markup o lenguaje de marcas de hipertexto, el lenguaje de programación que a traves de etiquetas se van definiendo los elementos, los cuales son interpretados por los navegadores, con las actualizaciones a traves del tiempo han permitido combinar propiedades de CSS3 y JAVASCRIPT para definir elementos básicos, En este sitio usaremos las etiquetas básicas para crear una página para la empresa HtmlDesign que contenga un menu, una imagen del tema y logo, el uso de etiquetas básicas como los son: Header para el emcabezado, nav para crear el menu, ul para crear una lista desordenada, li para crear un item con enlace, hr para colocar una línea, p para los párrafos, br para saltos línea entre otro para la elaboración recursos educativos digitales para dinamizar las estrategias de aprendizaje, pero antes que todo se debe realizar el mockup y subir una imagen del mismo.

ETIQUETAS BASICAS DE HTML

Las etiquetas sirven para estructurar el contenido de página HTML y contienen otras propiedades como atributos y contenido.

etiqueta Descripción
!DOCTYPE Define el tipo de docuemento
html Creando un documento HTML
Script Establecer instrucciones de programación JavaScript
div Contenedor que agrupa elementos HTML
title Titulo de la página
Head Define información hacerca del documento
Header Define la sección de encabezado del docuemnto
Body Define el cuerpo del documento
a Define un hipervínculo
img Define una imagen
nav Define un link de navegación
ul Define una lista desordenada
ol Define una lista ordenada
li Define un ítem de una lista
tr Define una fila en una tabla
td Define una celda en una tabla
h1...h6 Define encabezados o títulos
p Define un párrafo
section Define una sección de un documento
br Define un salto de línea
b Define texto en negrita
strong Define texto en negrita
article Define un artículo
footer Define el pie de página de un documento
